## Happy

My repository for learning 3D Rendering with OpenGL and SDL2

### Requirements

- C/C++ compiler
- OpenGL
- SDL2 (https://www.libsdl.org/download-2.0.php)
- Glad, an OpenGL and other spec generator loader (https://github.com/Dav1dde/glad)

#### Build and Run

```shell
$ make build
$ ./bin/happy
```
